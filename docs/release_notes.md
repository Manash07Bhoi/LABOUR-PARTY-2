# Release Notes

## Version: v1.0.1-hotfix-rc1

### Critical Hotfixes
- **Date Leakage Resolved:** Cross-day corruption is completely eliminated. `WorkId` allocation is strictly deterministic and dynamically bound to absolute Date+Session keys (`work_24_May_2026_Morning`).
- **Labour Identity Integrity:** Real typed labour names are guaranteed to persist exactly into Hive structures. The UI no longer falls back to arbitrary internal indexing ("Labour 1").
- **Add/Edit Stability:** Edit screens safely preserve parent elements. TripLabour subsets successfully support negative (deletion) actions by implementing dedicated safe-upsert cleanup parameters per `tripId`.
- **Dashboard Refresh Safety:** Exhaustive UI states perfectly intercept all loading flows. "Unexpected State in Dashboard" warnings generated by mid-action delays have been safely muted through integrated `WorkActionSuccess` mapping endpoints.
- **Search Implementations:** Live text filtering fully parses permutations.
